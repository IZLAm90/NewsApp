# News App
Project Overview
A modern news app that allows users to explore the latest news articles from various sources.

## Technologies Used
- Clean Architecture
- Modularity
- Flow and Coroutines
- Room Database
- Hilt Dependency Injection
- Dark Theme Support
- Retrofit for API calls
- MVVM Architecture
